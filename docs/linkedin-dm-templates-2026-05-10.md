## What never to send

- "Buy my product" — no DM ever sells a product on first touch
- Generic "let me know if you need anything" — provides zero specific value
- "Quick question" without the actual question — bait + switch reads as scammy
- Long message > 200 words — DMs over 200 words don't get read
- Messages that don't end with the sender's name — feels automated
